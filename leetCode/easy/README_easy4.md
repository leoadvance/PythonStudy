* [# 872. Leaf-Similar Trees (leafSimilar)](https://leetcode.com/problems/leaf-similar-trees/)  
题目需求对比两个二叉树最终端数据是否一致
要点：
    1. 遍历提取终端数据到list.
    2. 利用operator.eq()进行list内容一致性对比。
